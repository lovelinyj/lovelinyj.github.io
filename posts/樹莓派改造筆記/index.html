樹莓派改造筆記

April 22, 2022

    今天家裡多了一個raspberry pi 3B+
    馬上裝了raspberry os

    樹莓派安裝步驟

    所需工具

    1. 樹莓派
2. SD卡
3. 讀卡機
首先到官網上下載Raspberry Pi OS
[網頁連結](https://www.raspberrypi.com/software/)

    安裝完可以做什麼

    內網穿透

    Ngork (https://ngrok.com/)
    一款免費的內網穿透軟體

    wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-arm.tgz
    tar zxvf ./ngrok-v3-stable-linux-arm.tgz
    ngrok config add-authtoken [your token]
    ngrok http 80

    開機自動啟動Ngrok

    寫好一個script nano /home//autorun.sh

    /home//ngrok http 80

    然後在/etc/rc.local中新增開機執行指令

    /home//autorun.sh

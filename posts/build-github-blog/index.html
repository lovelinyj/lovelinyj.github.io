Build Github Blog

November 10, 2021

    When I build a github blog I need prepare something:

    1. a github account
    2. a github repository(use {yourname).github.ii)
    3. hugo

    use hugo to made new site

    hugo new site {yourname}.github.io cd {yourname}.github.io git init

    chose a beautiful theme

    try run hugo server

    push blog to github site
